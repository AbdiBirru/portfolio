# AbdiBuilds — Portfolio

Personal portfolio for Abdisa Birru, Full-Stack Developer.

## Stack
Next.js 14 (App Router), TypeScript, Tailwind CSS, Framer Motion, next-themes.

## Build
## Deployment
Auto-deployed via Vercel on every push to `main`.

## Structure
- `src/app` — routes, layout, metadata, sitemap/robots
- `src/components/sections` — page sections (Hero, About, Skills, Projects, Experience, Approach, GitHubStats, Blog, Contact)
- `src/components/ui` — reusable components (ProjectCard, RepoCard)
- `src/components/layout` — Navbar, Footer, theme/motion providers
- `src/components/icons` — self-contained brand icon SVGs
- `src/lib` — content data (projects, experience, socials)
